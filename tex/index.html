Local Positioning System Database (lpsdb)

Here is a database of datasets for calibrating local positioning systems. Typically there is a set of receivers r_i in R^3 and a set of senders s_j in R^3. Given some of the distance measurements d_ij = |r_i - s_j| determine r_i and s_j.

  1. Download the matlab code from https://github.com/kalleastrom/LocoPositioningSystemSlam
  2. The folder structure should be:
    • LocoPositioningSystemSlam
      • matlab
        • main.m
        • other matlab code and folders
      • data
        • savefiles (this folder is used to save intermediate results)
        • lpsgt (this folder contains ground truth for experiments)
        • lpsdb (this folder should contain the experiments you want to try. However, this folder is empty when you download it from github)
          • data1.mat (one experiment saved as a mat file)
          • data2.mat (another experiment saved as a mat file)
          • data3.mat
          • ...
      • tex
        • index.html (tex file explaining some of the math and algorithms)
        • LocoPositioningSystemSlam.tex (tex file explaining some of the math and algorithms)
  3. Download the raw sound files from this site http://vision.maths.lth.se/lpsdb/ and put the each experiment mat file in the folder data/sfsdb so that you get the right file structure.
  4. Run main.m
